# NutriTrack - Трекер калорийности и рациона питания

Клиентская часть веб-приложения для расчёта калорийности и планирования рациона питания.

## Технологии

- **HTML5** - семантическая разметка
- **CSS3** - переменные, адаптивная вёрстка (медиа-запросы), анимации
- **JavaScript ES6+** - модули, классы (ООП), async/await
- **Vite** - сборщик проекта
- **Chart.js** - интерактивные графики
- **Open Food Facts API** - бесплатная база продуктов питания (без ключа)
- **localStorage** - хранение данных в браузере
- **БЭМ** - методология именования CSS-классов
- **Git + GitHub Pages** - версионирование и публикация

---

## Быстрый старт

### Требования
- Node.js 18+ (скачать: https://nodejs.org)
- npm (идёт вместе с Node.js)
- Git

### Установка и запуск

```bash
# 1. Перейти в папку проекта
cd nutri-app

# 2. Установить зависимости
npm install

# 3. Запустить в режиме разработки
npm run dev
```

После этого откройте браузер: **http://localhost:3000**

## API

Используется Open Food Facts (https://world.openfoodfacts.org):
- Полностью бесплатный, без регистрации и API-ключей
- Содержит 3+ миллиона продуктов со всего мира
- Поиск: `GET /cgi/search.pl?search_terms=...&json=1`
- Продукт по штрихкоду: `GET /api/v0/product/{barcode}.json`

---

## Функциональность

| Раздел | Функции |
|--------|---------|
| **Поиск** | Поиск продуктов через API, просмотр калорий и БЖУ, добавление в дневник |
| **Дневник** | Приёмы пищи по дням, прогресс-бар калорий, навигация по датам |
| **Рацион** | Недельный планировщик, график потребления (Chart.js) |
| **Профиль** | BMR/TDEE по Миффлину-Сан Жеору, ИМТ, графики макросов |
